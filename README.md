# Hextra

Modern, responsive, batteries-included Hugo theme for creating beautiful static websites.

Forked from [imfing/hextra](https://github.com/imfing/hextra).

## Features

- Beautiful design powered by Tailwind CSS, with dark mode support
- Full-text search powered by FlexSearch
- Multi-language support (21 locales)
- Built-in shortcodes: cards, callouts, tabs, steps, and more
- LaTeX math, Mermaid diagrams, syntax highlighting
- SEO ready with Open Graph and Twitter Cards

See the [docs site](https://docs.jackieyang.ac.cn/) for usage and configuration.
